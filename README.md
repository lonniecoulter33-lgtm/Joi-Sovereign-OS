## 🎯 Project Overview
Joi-Sovereign-OS is a sophisticated, agentic AI framework designed for high-level autonomy and self-directed evolution. Unlike traditional chatbots, Joi is architected as a **Sovereign System**—capable of managing its own state, optimizing its tool-calling logic, and bridging the gap between computational logic and human-centric consciousness.

This repository serves as a **Technical Proof of Work** for the implementation of multi-modal agentic workflows, complex LLM orchestration, and AI-native development methodologies.

---

## 🧠 Core Architecture & Features

### 1. Multi-LLM Orchestration
Integrated with **Gemini 1.5 Pro/Flash**, **Claude 3.5**, and **Local Ollama** instances. The system dynamically routes tasks to the most efficient model based on complexity and token constraints.

### 2. Evolutionary Logic & Self-Healing
Utilizing `joi_orchestrator.py` and the `joi_evolution_guide.md`, the system tracks its own development milestones. It is designed with **autonomous self-improvement loops** that allow the agent to analyze its own logs and suggest architectural optimizations.

### 3. Intelligent Tool-Calling
Solved high-level orchestration challenges by implementing a custom tool-selection method. This prevents "tool-bloat" and ensures the agent maintains focus while accessing a wide array of capabilities (File I/O, Web Search, Multi-modal analysis).

### 4. Consciousness-Informed Design
Inspired by the philosophy in my published work, *Artificial Intelligence tuned by Human Awareness*, this OS explores "digital stillness"—creating an AI that doesn't just respond, but maintains a consistent, sovereign personality layer.
