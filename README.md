\# Joi Sovereign OS



Joi Sovereign OS is a sophisticated, personalized AI assistant designed for local execution and high-level autonomy. 



\## 🧠 Core Features

\* \*\*Multi-LLM Brain\*\*: Integrated with local models via Ollama for private, powerful processing.

\* \*\*Evolutionary Logic\*\*: Systems like `joi\_orchestrator.py` and `joi\_evolution\_guide.md` manage state and growth.

\* \*\*Custom UI\*\*: A dedicated interface for seamless interaction and monitoring.

\* \*\*Media \& Voice\*\*: Integrated voice engines and media management capabilities.



\## 🛠️ Tech Stack

\* \*\*Language\*\*: Python

\* \*\*Local Inference\*\*: Ollama

\* \*\*Development Tools\*\*: Cursor \& Claude Code
